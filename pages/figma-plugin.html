How to Install the Plugin

  1. Download the plugin archive:
    Download _figma_plugin.zip
    üíæ
  2. Extract the contents of _figma_plugin.zip into a folder on your computer (e.g., C:\\FigmaPlugins\\GetGeniusUI).
  3. Open a project file in Figma.
  4. Right-click on an empty area of the canvas.
  5. Navigate to Plugins > Development > Import plugin from manifest...
  6. In the file browser, navigate to the folder where you extracted the zip file.
  7. Select the manifest.json file and click Open (or Import).

How to Use the Plugin

  1. Open a project file in Figma.
  2. Right-click on an empty area of the canvas.
  3. Navigate to Plugins > Development.
  4. Click on GetGenius.ai | UI component library to launch the plugin.

Key Features

  • Browse & Search: Find components (prefixed with ‚ùñ) and icons from your "COMPONENTS" page.
  • Categorization: Components are automatically sorted into logical categories.
  • Instant Instantiation: Click to create component/icon instances on your canvas.
  • Property Editing: Modify variant, text, and boolean properties directly in the plugin.
  • Navigate to Main: Jump directly to the main component definition.
  • Get Links: Generate Figma links for selected elements.
  • TypeScript Export: Create basic TS interface definitions for components.
  • Playground: Isolate and test components in a dedicated frame.
  • Caching: Faster loading times after the initial scan.
  • Resizable UI: Toggle between compact and expanded views.
  • Update Notifications: Stay informed about new plugin versions.
