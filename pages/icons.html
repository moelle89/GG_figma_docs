Icons

Using Ionicons in your designs

This project uses Ionicons, an open-source icon library created by Ionic and released under the MIT license. The icons are designed for use in web, iOS, Android, and desktop applications.

Icon Usage

Basic Implementation

  • Include the Ionicons script in your HTML head
  • Use the ion-icon element with a name attribute
  • Icons automatically scale with font-size
<ion-icon name="heart-outline"></ion-icon>

Style Variations

  • Outline style: name="icon-outline"
  • Filled style: name="icon"
  • Sharp style: name="icon-sharp"
