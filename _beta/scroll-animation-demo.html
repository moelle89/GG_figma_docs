Text Fill Animation

This paragraph uses the has-opacity class. As you scroll, each word will fade in gradually, creating a smooth reveal effect. The animation is triggered when the element reaches a certain point in the viewport.

Light on Dark Background

The same animations work on dark backgrounds too. Notice how the text fill color changes to white instead of black. This creates a consistent visual effect regardless of the background color.

Block Title Animation

The block-title variant treats each word as a separate block, which can create interesting staggered effects when combined with the text fill animation. Try scrolling slowly to see how each word reveals independently.

How It Works

These animations use GSAP and ScrollTrigger to create scroll-based text reveal effects. The JavaScript splits text into spans and animates properties like background-size and opacity as you scroll through the page.
